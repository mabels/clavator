# clavator

[![Build Status](https://travis-ci.org/mabels/clavator.svg?branch=master)](https://travis-ci.org/mabels/clavator)

## Dependencies

* gcc
* autoconf
* nodejs 6.x

## Getting started

In a first terminal:

```shell
npm install
npm run build:watch
```

In a second terminal:

```shell
npm start
```

Then open http://localhost:8888 in your browser.
